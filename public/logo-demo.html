üìö Quick Reference

Logo Files

  • /public/icon.svg - Vector logo (recommended)
  • /public/icon.png - Raster logo (512x512px)

Logo Component

  • /app/views/shared/_logo.html.erb - Reusable partial
  • /app/assets/stylesheets/design-system/logo.css - Styles

Usage Examples

  • <%= render 'shared/logo' %> - Default logo
  • <%= render 'shared/logo', size: 'sm' %> - Small
  • <%= render 'shared/logo', size: 'lg' %> - Large
  • <%= render 'shared/logo', text: false %> - Icon only
  • <%= render 'shared/logo', link: false %> - Non-clickable

Where to Add

  • Sidebar: app/views/shared/_sidebar.html.erb
  • Topbar: app/views/shared/_topbar.html.erb
  • Mobile: Use <%= render 'shared/brand_bar' %>
  • Login: app/views/sessions/new.html.erb
